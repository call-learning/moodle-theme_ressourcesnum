Instructions to import Moment into CLTools:

1. Download the latest release from https://github.com/glidejs/glide
2. copy 'https://github.com/glidejs/glide/tree/master/dist/glide.js' into 'amd/src/glide.js'
   Add /* eslint-disable */ in the beginning
3. copy 'https://github.com/glidejs/glide/tree/master/dist/css/*' into 'style/*'
   Add /* stylelint-disable */ in the beginning
